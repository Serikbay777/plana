# Паркинг и стоянки автомобилей РК

Документ обобщает действующие в Республике Казахстан нормативные требования по проектированию автомобильных стоянок и подземных паркингов в составе жилой и общественной застройки. Цифры предназначены для использования в AI-генераторе концептуальных site plan и подземных паркингов Plana.

| СНиП РК 3.01-01-2008 | «Градостроительство. Планировка и застройка городских и сельских населённых пунктов» — таблица 11.8 (разрывы), нормы расчёта мест |

Внутренний приказ Комстроя № 54-НҚ от 27.04.2021 относит ряд норм по машино-местам к рекомендательным («необязательного применения») для малогабаритного жилья — это объясняет различную плотность парковки в разных ЖК.

## Нормы машино-мест по типу объекта

Базовый расчёт ведётся по СНиП РК 3.01-01-2008 и СП РК 3.03-105-2014. Для жилья применяется классификация Colliers/QazBDS по классам.

| Объект | Норма машино-мест |
|---|---|
| Жильё, класс премиум / бизнес | 2,0 м/м на квартиру |
| Жильё, класс комфорт | 1,0 м/м на квартиру |
| Жильё, класс эконом | 0,5 м/м на квартиру (или не нормируется для малогабаритного) |
| Гостевые места (доп. к жильцам) | 10–15 % от общего числа квартирных мест |
| Офисы класс A | 1 м/м на 60–80 м² арендуемой площади |
| Офисы класс B/C | 1 м/м на 100–120 м² |
| Торговые центры | 3–5 м/м на 100 м² торговой площади (≈ 1 м/м на 25–35 м²) |
| Магазины районного значения | 1 м/м на 50–80 м² торгового зала |
| Гостиницы 4–5* | 1 м/м на 3–4 номера + надбавка для ресторанов/конференц-залов |
| Гостиницы 2–3* | 1 м/м на 6–10 номеров |
| Бизнес-центры с конференц-залами | 1 м/м на 5 посетителей зала |
| Поликлиники | 1 м/м на 10–15 посещений в смену |
| Больницы (стационар) | 1 м/м на 10 коек + персонал |
| Школы, детсады | 2–5 мест у входа (в основном кратковременная high-turnover стоянка) |
| Спортсооружения, ТРЦ, кинотеатры | 1 м/м на 5–7 зрительских мест |
| Рестораны | 1 м/м на 5–8 посадочных мест |
| Промышленные объекты | По числу работающих в макс. смене + автопарк |

Для жилых комплексов, состоящих из квартир до 35–40 м², застройщик вправе применять нормы как рекомендательные — это закреплено приказом № 54-НҚ.

## Размеры машино-места

| Параметр | Значение |
|---|---|
| Стандартное машино-место (с зазорами) | 2,5 × 5,5 м (СН РК 3.03-05-2014) |
| Габарит самого автомобиля (без зазоров) | 1,8 × 4,4 м (легковой M1) |
| Минимальные зазоры | 0,5 м между авто, 0,7 м до стены |
| Машино-место для МГН (колясочник) | 3,6 × 6,0 м (СН РК) или ширина не менее 3,5 м |
| Грузовое место (категория N1) | 3,0 × 6,5 м |
| Грузовое место (N2/N3) | 3,5 × 8,5–13,0 м |
| Микроавтобус | 2,8 × 6,5 м |
| Высота помещения | не менее 2,0 м, и ≥ высоты авто + 0,2 м |
| Высота над МГН-местом | не менее 2,1 м |

Российский СП 113.13330 допускает 2,3 × 5,0 м как минимум для самого автомобиля без зазоров — это часто видят в проектной документации, но казахстанская норма требует именно 2,5 × 5,5 м с учётом зазоров. Для генератора используем 2,5 × 5,5 м как «канонический» прямоугольник места.

## Проезды и аэрации

| Конфигурация | Минимальная ширина проезда |
|---|---|
| Перпендикулярная парковка (90°), двусторонний проезд | 7,0 м |
| Перпендикулярная парковка (90°), односторонний | 6,0 м |
| Парковка под углом 60° | 5,5 м |
| Парковка под углом 45° | 4,5 м |
| Параллельная парковка | 3,5 м (одностор.) / 6,0 м (двустор.) |
| Прямолинейная рампа, односторонняя | 3,5 м |
| Прямолинейная рампа, двусторонняя | 6,0 м (с разделительной полосой 0,3 м) |
| Криволинейная рампа, односторонняя | 3,8 м (по внутр. радиусу) |
| Внутр. радиус криволинейной рампы | не менее 7,0 м (по оси полосы) |
| Уклон закрытых рамп прямолинейных | до 18 % |
| Уклон закрытых рамп криволинейных | до 13 % |
| Уклон открытых рамп | до 10 % |
| Поперечный уклон | не более 6 % |
| Тротуар при рампе | не менее 0,8 м, отделён бортом 0,1 м |
| Проезд для пожарной техники | не менее 4,0 м (свободная ширина), 4,5 м высота |

Внутриквартальные проезды жилой застройки: 3,5 м (одностор.) / 5,5–6,0 м (двустор.). При организации парковки вдоль проезда минимальная общая ширина увеличивается до 6,5 м (одностор.) и 9,5 м (двустор.).

## Подземный паркинг — особые требования

| Требование | Значение |
|---|---|
| Максимальное число подземных уровней | 5 |
| Высота этажа (чистая) | ≥ 2,0 м, и ≥ высоты авто + 0,2 м |
| Конструктивная высота этажа | обычно 3,0–3,3 м (с балками, коммуникациями) |
| Высота под балками/коробами | ≥ 2,0 м |
| Число рамп на этаж | до 100 м/м — 1 однопол.; до 1000 — 1 двупол. или 2 однопол.; > 1000 — 2 двупол. |
| Эвакуация: длина пути до выхода | ≤ 40 м (≤ 20 м в тупиковой части) |
| Расстояние между эвакуац. выходами | не более 40 м, не менее 2 выходов с этажа |
| Огнестойкость несущих конструкций | REI 150 (I степень) |
| Площадь пожарного отсека | до 3000 м² на этаж (одноуровневый отсек) |
| Автоматическое пожаротушение | обязательно для всех подземных стоянок независимо от этажности |
| Дымоудаление | обязательно; струйная (jet) вентиляция допускается с отдельным расчётом |
| Приточно-вытяжная вентиляция | расчёт по СО/CO₂; нагрев на 5 °C для отапливаемых |
| Газобаллонные авто (СУГ/КПГ) | запрещено хранить ниже уровня земли |
| Индивидуальные боксы | в подземной части не допускаются |
| Расположение въездов от окон жилья | ≥ 15 м |
| Объекты обслуживания, торговли в зоне хранения | запрещены |

Если стоянка имеет рампу с подогревом покрытия (актуально для Астаны/Караганды/Кокшетау), допустимый продольный уклон может приниматься как для закрытой — до 18 %.

## МГН-парковка

| Параметр | Значение |
|---|---|
| Доля МГН-мест | не менее 10 % от общего числа, минимум 1 место |
| Из них для колясочников | не менее 5 % от общего числа (часть от 10 %) |
| Размер места | 3,6 × 6,0 м (или ширина ≥ 3,5 м) |
| Разметка | желтая/синяя, символ ISA, габарит обязателен |
| Расстояние до доступного входа | ≤ 50 м (до жилого — ≤ 100 м) |
| Высота над МГН-местом (подземный) | ≥ 2,1 м |
| Уклон поверхности | не более 2 % (поперечный и продольный) |
| Тактильное покрытие на пути | обязательно от места к входу |

Для общественных зданий (поликлиники, акиматы, ТРЦ) требуется выделять минимум 4 МГН-места при общем числе свыше 50.

## Бытовые разрывы (расстояния от зданий)

Таблица 11.8 СНиП РК 3.01-01-2008. Расстояние измеряется от окон жилых/общественных зданий или границ участка ДОУ/школы/больницы до стены гаража или границы открытой стоянки.

| Тип здания / число м/м на стоянке | ≤10 | 11–50 | 51–100 | 101–300 |
|---|---|---|---|---|
| Жилые дома (до окон) | 10 м | 15 м | 25 м | 35 м |
| Жилые дома, торец без окон | 10 м | 10 м | 15 м | 25 м |
| Общественные здания | 10 м | 10 м | 15 м | 25 м |
| Школы, детсады, школы-интернаты | 15 м | 25 м | 25 м | 50 м |
| Больницы со стационаром | 25 м | 50 м | по согласованию с СЭС | по согл. |

Дополнительно:
- Въезд/выезд из подземного гаража — не ближе 15 м от окон жилых домов, рабочих помещений, площадок ДОУ, школ, больниц.
- Стоянка на 300+ м/м должна согласовываться с органами санэпиднадзора отдельно.
- Открытые стоянки на придомовой территории с учётом гостевых на 10 м/м не нормируются как «гараж».

## Что важно для AI-генератора site plan

1. **Канонический модуль м/м: 2,5 × 5,5 м.** Использовать как базовый прямоугольник для расстановки.
2. **Сетка проезд+ряд+ряд = 2,5 + 5,5 + 7,0 + 5,5 + 2,5 = 23 м** (двусторонняя обвязка двух рядов с двусторонним проездом). Это эталонный шаг для подземного паркинга.
3. **Норма мест:** для жилья по умолчанию 1,0 м/м на квартиру (комфорт-класс) + 15 % гостевых; класс премиум — 2,0; эконом — 0,5.
4. **МГН:** автоматически выделять 10 % мест ближе к входу/лифтам с увеличенной шириной 3,6 м.
5. **Разрывы:** при размещении наземной парковки на site plan соблюдать таблицу 11.8 — поставить динамические буферы 10/15/25/35 м в зависимости от ёмкости относительно фасадов жилья.
6. **Подземный паркинг:** базовая высота этажа 3,0–3,3 м, максимум 5 уровней, 1 рампа до 100 м/м, далее +1 рампа. Площадь пожарного отсека ≤ 3000 м².
7. **Рампа:** уклон до 18 % (закрытая, прямая), ширина одностор. 3,5 м, двустор. 6,0 м; внутр. радиус ≥ 7 м.
8. **Длина пути эвакуации:** не размещать места дальше 40 м от ближайшей лестничной клетки в подземной части (20 м в тупике).
9. **Велопарковки:** 1 место на 1–2 квартиры (рекомендация по аналогии с РФ/ЕАЭС, в нормах РК прямой нормы нет, но действует ГОСТ 33150-2014). Шаг 0,6–0,8 м между велостойками.
10. **Отличия от СП 113.13330 (РФ):** в РК размер места **2,5 × 5,5 м с зазорами** (в РФ — 2,3 × 5,0 м без зазоров); МГН **3,6 × 6,0 м** (в РФ — 3,6 × 6,0 м или ширина ≥ 3,5 м — совпадает); таблица разрывов в РК идёт по СНиП РК 3.01-01-2008, в РФ — по СанПиН 2.2.1/2.1.1.1200-03. РК допускает применение российских СП в качестве пособия при отсутствии прямой нормы.
