# Доступная среда (МГН) РК

Маломобильные группы населения (МГН) — лица с ограничениями подвижности (постоянными: пользователи кресел-колясок, инвалиды по зрению/слуху; временными: люди с детскими колясками, тяжёлым багажом, после травм). Раздел сводит требования к проектированию жилых и общественных зданий в Казахстане применительно к задаче AI-генерации планировок.

## Пандусы и подъёмы

| Параметр | Значение |
|---|---|
| Стандартный продольный уклон (новое строительство) | **не круче 1:20** (5 %) |
| Допустимый уклон (реконструкция, капремонт, временные сооружения) | **до 1:12** (8 %), при подъёме ≤ 0,5 м и длине марша ≤ 6,0 м |
| Максимальный уклон при перепаде ≤ 0,2 м | 1:10 (10 %) — короткий аппарель |
| Поперечный уклон | ≤ 1–2 % |
| Ширина марша между поручнями | **0,9–1,0 м** (одностороннее движение) |
| Длина одного марша | ≤ 9,0 м |
| Горизонтальная площадка в начале/конце | ≥ 1,5 × 1,5 м (при повороте — обеспечить разворот коляски) |
| Бортик по внешнему боковому краю | ≥ 0,05 м (5 см) |
| Поручни — высота | 0,7 и 0,9 м (двойные); для детей — 0,5 м |
| Диаметр поручня | 32–50 мм |
| Свесы поручней за конец марша | ≥ 0,3 м |
| Когда нужен пандус | при перепаде высоты на пути движения МГН > 0,014 м (14 мм) |

Поверхность пандуса — твёрдая, нескользкая; контрастная маркировка крайних ступеней/кромок.

## Дверные проёмы

| Расположение | Минимальная ширина в свету |
|---|---|
| Главный вход в здание (МГН-маршрут) | **1,2 м**; при двустворчатой — рабочая створка ≥ 0,9 м |
| Двери на путях МГН (коридор, лестничный узел) | **0,9 м** в свету |
| Внутриквартирные двери (жилые комнаты) | ≥ 0,9 м (рек.); допускается 0,8 м |
| Дверь в санузел квартиры (адаптируемой) | ≥ 0,9 м, **открывание наружу** |
| Дверь подсобной кладовой/гардероба | ≥ 0,7 м (исключение) |
| Порог в дверях на пути МГН | **≤ 0,014 м (14 мм)**; для адаптируемых квартир — ≤ 5 мм или без порога |
| Усилие открывания недоводчиковой двери | ≤ 50 Н |
| Глубина тамбура | ≥ 1,8 м (при ширине ≥ 2,2 м) |

Запрещены: вращающиеся двери и турникеты на путях МГН без альтернативного прохода; ступени в дверном проёме.

## Лифты для МГН

Лифт **обязателен**, если этаж входа в здание выше отметки + 13,2 м (~4 этажа), а также во всех общественных зданиях с уровнями, обслуживающими МГН. В жилых зданиях ≥ 2 этажей с квартирами для инвалидов-колясочников — лифт обязателен.

| Тип кабины | Внутренние размеры | Дверь | Применение |
|---|---|---|---|
| Стандартная МГН (тип 1 по ГОСТ 33652) | **1,1 × 1,4 м** (ш × гл) | ≥ 0,9 м | Минимум для коляски без сопровождающего |
| Расширенная МГН (тип 2) | **1,1 × 2,1 м** | ≥ 0,9 м | Коляска + сопровождающий, носилки |
| Для эвакуации МГН / больничный | **1,5 × 2,1 м** или **1,7 × 1,5 м** | ≥ 1,2 м | Жилые ≥ 9 эт.; больницы; эвакуация |

Требования к кабине: поручень на высоте 0,9 м (доп. 0,7 м), панель управления на высоте 0,85–1,1 м с шрифтом Брайля, звуковое и визуальное информирование о прибытии и этаже. Перед лифтом — площадка ≥ 1,6 × 1,6 м.

## МГН-парковка

| Параметр | Значение |
|---|---|
| Доля мест для инвалидов | **≥ 10 %** от общего количества мест (Закон РК; дробное ≥ 0,5 округляется вверх) |
| Из них для инвалидов-колясочников | ≥ 5 % (но не менее 1 места) |
| Размер машино-места МГН (стандарт) | **3,6 × 6,0 м** (ширина увеличена за счёт зоны выгрузки 1,2 м сбоку) |
| Размер для микроавтобуса с подъёмником | **3,6 × 8,0 м** |
| Расстояние до входа в здание (общественные) | ≤ 50 м |
| Расстояние до входа в жилой дом | ≤ 100 м |
| Маркировка | Знак «Инвалид» (вертикальный + на покрытии), контрастная разметка |
| Покрытие | Твёрдое, ровное; уклон ≤ 2 % |

Места — бесплатные на объектах социальной инфраструктуры, транспортных узлах, в кварталах жилой застройки.

## Доступные квартиры — внутренние требования

Для жилых многоквартирных домов — **минимум 5 % квартир** (по серии проектов) должны быть адаптируемыми / доступными для инвалидов-колясочников; допускается размещать на первом этаже с прямым доступом без лифта.

| Зона | Параметр |
|---|---|
| Прихожая (для разворота коляски) | свободный диаметр **Ø 1,4 м** или квадрат **1,5 × 1,5 м** |
| Ширина внутриквартирного коридора (одностороннее движение) | ≥ 1,15 м (рек. 1,5 м) |
| Ширина коридора при встречном движении | ≥ 1,8 м |
| Локальное сужение | до 0,85 м (≤ 0,5 м длиной) |
| Зона перед мебелью / прибором | подход ≥ 0,9 м; при повороте на 90° — ≥ 1,2 м |
| Высота прохода до низа выступающих конструкций | ≥ 2,1 м |
| Высота столешниц кухни (адаптируемая) | **0,8–0,85 м**, с подкатным пространством под столом 0,7 × 0,5 м |
| Высота электророзеток / выключателей | 0,4–1,1 м от пола |
| Зона у входной двери (внутри) | ≥ 1,5 × 1,5 м |
| Балкон / лоджия | ширина прохода ≥ 0,9 м, без порога > 14 мм |

Лестница в двухуровневой квартире — недопустима как единственный путь; нужен подъёмник или планировка в одном уровне.

## Санузлы — размеры

### В квартирах для МГН (совмещённый)

| Параметр | Значение |
|---|---|
| Минимальные внутренние габариты | **2,2 × 2,2 м** (свободная зона + Ø 1,4 м для разворота) |
| Раздельный туалет (МГН) | **1,65 × 1,8 м** (мин.) |
| Зона сбоку от унитаза (для пересадки) | ≥ 0,8 м (рек. 0,9 м) |
| Зона перед унитазом | ≥ 0,6 м |
| Высота унитаза от пола | 0,45–0,5 м |
| Поручни откидные/стационарные | высота 0,7–0,85 м, длина 0,6–0,8 м, Ø 30–50 мм |
| Душ без поддона (низкий борт) | ≥ 1,5 × 0,85 м, борт ≤ 0,025 м |
| Раковина | высота 0,8 м; подколенное пространство 0,7 × 0,5 м |
| Зеркало | нижняя кромка ≤ 0,9 м от пола |

### В общественных зданиях

| Тип кабины | Размеры |
|---|---|
| Универсальная МГН-кабина (с унитазом и раковиной) | **1,7 × 2,2 м** (или 1,8 × 2,2 м) |
| Доступная кабина в общем санузле | **1,65 × 2,2 м** |
| Дверь | ≥ 0,9 м, открывание **наружу**, замок изнутри без ключа |
| Кнопка экстренного вызова | на высоте 0,3 и 0,9 м от пола |

## Тактильные направляющие, контраст

| Элемент | Норматив |
|---|---|
| Тактильно-контрастные предупреждающие указатели перед лестницей/пандусом/проездом | полоса 0,5–0,6 м, на расстоянии 0,8 м от препятствия |
| Высота тактильных рифов на покрытии | ≤ 5 мм (не создаёт препятствие коляске) |
| Контрастные полосы на ступенях | первая и последняя ступень, ширина 0,05–0,065 м, контраст ≥ 60 % |
| Контраст дверного полотна / ручки относительно стены | ≥ 60 % по светлоте |
| Маркировка стеклянных дверей | две полосы на высоте 1,0–1,2 м и 1,5–1,7 м |
| Шрифт Брайля и крупный шрифт | у дверей помещений общественных зданий, в лифтах |
| Звуковое информирование | в лифтах, на входах общественных зданий |
| Уровень освещённости на путях МГН | повышенный на 20–30 % относительно базовой нормы |

## Что важно для AI-генератора планировок (где появляется МГН-зона)

1. **Маршрут МГН (golden path).** От парковки/входа на участке → главный вход → лифт → квартира. Проверять ширину ≥ 1,5 м, отсутствие ступеней, пороги ≤ 14 мм. Пандус автоматически вставляется при перепаде > 14 мм.
2. **Бюджет 5 %.** В каждом подъезде/блоке резервировать ≥ 5 % квартир под МГН-планировку, предпочтительно на первом этаже. Помечать атрибутом `accessible: true`.
3. **«Круг МГН».** В адаптируемой квартире валидатор должен находить **свободную окружность Ø 1,4 м** в прихожей, кухне (зона у плиты/мойки), санузле, спальне (у изголовья кровати).
4. **Санузел МГН — отдельный шаблон.** Не просто «увеличить» обычный, а вызывать отдельный template `bath_mgn` с минимальными габаритами 2,2 × 2,2 м (совмещённый) или 1,65 × 1,8 м (туалет).
5. **Двери.** Все двери на МГН-маршруте — ширина ≥ 0,9 м. Двери санузлов адаптируемых квартир открываются **наружу**. Тип «купе/раздвижная» допустим — при том же требовании к ширине просвета.
6. **Лифт.** При высоте здания > 13,2 м или при наличии адаптируемых квартир выше 1 этажа — резервировать шахту ≥ 1,8 × 2,2 м под кабину типа 2 (1,1 × 2,1 м). Площадка перед лифтом ≥ 1,6 × 1,6 м.
7. **Парковка на участке.** Если генератор размещает паркинг — выделять ≥ 10 % мест 3,6 × 6,0 м, на дистанции ≤ 100 м (жильё) / ≤ 50 м (соцобъекты) от входа, с прямым безбарьерным маршрутом.
8. **Пандус как объект.** Если у входа есть перепад h, длина пандуса L = h × 20 (при 1:20) или h × 12 (при 1:12 — для реконструкции). При L > 6 м — площадка отдыха 1,5 × 1,5 м.
9. **Высоты в адаптируемых квартирах.** Параметризовать: розетки 0,4–1,1 м; столешница кухни 0,8–0,85 м с пустотой под коленями; подоконник доступного окна ≤ 0,85 м.
10. **Контрастная разметка / тактиль.** В режиме «общественные пространства» (вестибюль, лифтовый холл, лестница) автоматически добавлять предупреждающие полосы перед перепадами и контраст «дверь vs стена» — это влияет на визуальный рендер плана и спецификацию отделки.
11. **Эвакуация.** Зона безопасности МГН (refuge) на каждом этаже многоэтажки: 2,4 × 2,4 м у эвакуационного лифта/лестницы, с системой двусторонней связи.
12. **Ошибки-«красные»** (fail-fast): пороги > 14 мм, лифт < 0,9 м двери, санузел МГН < 1,65 × 1,8 м, парковка МГН < 10 %, пандус > 1:12.
